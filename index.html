04 · What you need to build this

Five components.
Then it runs itself.

Each one takes less than an hour to set up. Follow them in order and you have a working agent by the end of the afternoon.

01
WhatsApp Business + Automation
30 to 45 min

The pipe that sends and receives messages automatically.

  1. Go to business.whatsapp.com and create a WhatsApp Business account for your business number.
  2. Sign up for Make at make.com. The free plan works to start.
  3. Inside Make, search for the WhatsApp Business module and connect your account using the API key WhatsApp provides.
  4. Test the connection by sending a message to yourself from inside Make.
02
Airtable Base · Your Logging System
20 min

Every conversation, sentiment, and outcome captured in one place you can audit.

  1. Go to airtable.com and create a free account.
  2. Create a new base called Review Agent.
  3. Add these columns: Customer Name, Phone Number, Date of Contact, Sentiment (positive / neutral / negative), Feedback Summary, Review Link Shared (yes/no), Next Steps, Do Not Contact (checkbox).
  4. Inside Make, add an Airtable module and connect it using your Airtable API key, found under your account settings.
  5. Every time the agent finishes a conversation, Make writes a row to this table automatically.
03
Your Google Review Link
5 min

The only link your customers will ever see.

  1. On your phone or desktop, search your business name on Google.
  2. Click your business listing, then click Reviews.
  3. Click Get more reviews and copy the link Google gives you.
  4. Paste it into your Make scenario wherever the agent shares the review link.
04
AI Model via API · Claude or GPT-4
45 to 60 min

The brain that reads the customer's message, asks the right questions, and drafts the review.

  1. Go to console.anthropic.com or platform.openai.com and create an account.
  2. Generate an API key from your account dashboard.
  3. Inside Make, add an HTTP module — not a specific WhatsApp or Airtable module, just a plain HTTP request.
  4. Set the URL to the API endpoint. Anthropic and OpenAI both show this in their docs.
  5. Paste your API key into the authorisation header.
  6. Write your agent instructions in the prompt field. This is where you tell the AI how to respond, what questions to ask, and how to draft the review.
  7. The AI reads the customer's message, generates a reply, and sends it back through Make to WhatsApp.
05
Post-Booking Trigger
20 to 30 min

The signal that fires the agent the moment an appointment finishes.

  1. If you use Calendly: inside Make, add a Calendly module and trigger when a booking is marked complete.
  2. If you use Google Calendar: add a Google Calendar module and trigger when an event ends.
  3. If you use a CRM like HubSpot or GoHighLevel: use their Make module and trigger on appointment completed or the equivalent status.
  4. Connect this trigger to the WhatsApp module so the message fires automatically after the appointment finishes.
  5. Set a delay of 30 to 60 minutes so the message arrives once the customer has had time to settle, not the second they walk out.

Total build time, first-timer: 3 to 4 hours.
